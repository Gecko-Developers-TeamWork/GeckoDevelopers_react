per creare il progetto react:
- npx create-react-app nome-repo
- cd nome-repo
- npm start  (o: yarn start)

in caso di errore a npm start si fa:
- npm update

per aggiungere il scss:
- npm instal sass (o: yarn add sass)
- rinomina i file in .scc (tipo App.css in App.scss)
- nei file .js modifichi l'import da .css a .scss
